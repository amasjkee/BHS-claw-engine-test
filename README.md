# BHS Claw Engine Test

Полнофункциональный проект ядра игрового движка с ECS архитектурой, продвинутой физикой и графическим интерфейсом.

## 🎯 Реализованные функции

### ✅ Базовая задача
- **Модульная архитектура** - четкое разделение на слои (Core, Scene, Physics, ECS, UI)
- **ECS интеграция** - полная интеграция с LeoECS Lite
- **Физическая симуляция** - движение шарика с отражением от стен
- **Система коллизий** - детекция столкновений с выводом ID стены
- **Скорость < радиуса** - соблюдение требований ТЗ

### 🚀 Дополнительные задачи
- **Произвольные стены** - поддержка стен под любым углом
- **Continuous Collision Detection (CCD)** - защита от "перелета" через стену
- **Avalonia UI** - графическое представление в реальном времени
- **Двойной режим** - консольный и GUI интерфейс

## 🏗️ Архитектура

```
src/
├── Core/                    # Базовые компоненты
│   ├── Math/               # Vector2, MathHelper

│   ├── SceneObject.cs     # Базовый класс
│   ├── Wall.cs           # Стена (отрезок)
│   ├── Ball.cs           # Шарик (окружность)
│   └── Scene.cs          # Контейнер сцены
├── Physics/               # Физические расчеты
│   ├── CollisionDetector.cs           # Базовый детектор
│   ├── AdvancedCollisionDetector.cs  # Продвинутый детектор
│   ├── PhysicsCalculator.cs          # Базовые расчеты
│   └── AdvancedPhysicsCalculator.cs  # Продвинутые расчеты
├── ECS/                   # ECS система
│   ├── Components/        # ECS компоненты
│   ├── Systems/          # ECS системы

## 🎮 Режимы запуска

### Консольный режим

```
- Автоматическая симуляция
- Детальный вывод коллизий
- Отладочная информация

### GUI режим

## 🔧 Технические особенности

### Продвинутая физика
- **Произвольные углы стен** - корректное отражение от диагональных стен
- **CCD защита** - предотвращение "туннелирования" через стены
- **Коррекция позиции** - автоматическая коррекция при проникновении
- **Детальная отладка** - информация о точках столкновения и нормалях

### ECS архитектура
- **Компоненты**: Transform, Velocity, Collider, SceneObjectLink
- **Системы**: Movement, AdvancedCollision
- **Синхронизация** - автоматическая синхронизация между ECS и Scene объектами

### Модульность
- **Разделение ответственности** - каждый слой независим
- **Тестируемость** - физика отделена от ECS
- **Расширяемость** - легко добавить новые типы объектов

## 🎮 Инструкции по использованию GUI

### Основные функции:
1. **Запуск симуляции** - нажмите "Старт" для начала физической симуляции
2. **Добавление шариков** - кликните мышкой в любом месте игрового поля
3. **Очистка** - кнопка "Очистить шарики" удаляет все шарики
4. **Сброс** - кнопка "Сброс" пересоздает сцену с диагональной стеной
5. **Остановка** - кнопка "Стоп" останавливает симуляцию

### Особенности:
- **Случайная скорость** - каждый новый шарик получает случайную скорость
- **Произвольные стены** - диагональная стена для демонстрации сложной физики
- **CCD защита** - предотвращение "туннелирования" через стены
- **Реальное время** - визуализация обновляется 60 раз в секунду
